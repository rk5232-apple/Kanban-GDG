# ZenBoard — GDG SRM Recruitment 2026

## Live Demo
🔗 https://kanban-gdg-pi.vercel.app/

## Features
- Drag & Drop (Desktop + Mobile touch support)
- Dark / Light Theme toggle
- localStorage Persistence
- Add / Edit / Delete Tasks
- Priority Labels (High / Medium / Low)
- Donut Chart Analytics (Task Distribution)
- Responsive Design (Mobile vertical scroll)
- Custom Columns (Add / Rename / Delete)

JavaScript (ES6+) — Drag & Drop API, Touch Events
localStorage — Data Persistence
